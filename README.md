# Learning Management Dashboard

- Responsive learning dashboard for course discovery, progress tracking, quizzes, and tasks.
- Built as a front-end project using React, JavaScript, Tailwind CSS, and Vite.
- Live demo: https://harshit1330.github.io/Learning-Management-Dashboard/

## Features

- Course cards with search and category filters.
- Progress tracking dashboard UI.
- Interactive quiz with score calculation.
- Task manager with `localStorage`.
- Dark/light mode support.
- Reusable React components.

## Tech Stack

- React
- JavaScript
- Tailwind CSS
- Vite
- localStorage

## Run Locally

```bash
npm install
npm run dev
```
