# Learning Management Dashboard

* A beginner-friendly front-end project built using React, JavaScript, and Tailwind CSS.
* This project is a simple learning dashboard where users can explore courses, track progress, take a quiz, manage tasks, and switch between dark and light mode.

## Features

* Responsive dashboard layout
* Course cards with search and category filter
* Progress tracking UI
* Interactive quiz with score calculation
* Task manager with localStorage
* Dark/light mode
* Clean and reusable React components

## Tech Stack

* React
* JavaScript
* Tailwind CSS
* Vite
* localStorage

## How to Run

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Bullet Points

* A responsive learning management dashboard using React, JavaScript, and Tailwind CSS.
* Implemented course search, category filters, progress tracking, quiz scoring, and a localStorage-based task manager.
* Created reusable React components for course cards, dashboard stats, quiz interface, and task management.
* Designed a mobile-friendly UI with dark mode and clean front-end styling practices.
